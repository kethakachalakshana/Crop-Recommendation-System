# 🌱 Crop Recommendation System using Machine Learning

## 📌 Project Overview
This is a Data Science project that uses Machine Learning to recommend the most suitable crop to plant based on soil and environmental conditions. This solution aims to assist the agriculture sector in making data-driven decisions.

## 📊 Dataset Features
The model analyzes the following conditions:
* Nitrogen (N), Phosphorus (P), Potassium (K) levels
* Temperature, Humidity, and pH value of the soil
* Average Rainfall

## 🛠️ Technologies Used
* **Language:** Python
* **Libraries:** Pandas, Scikit-Learn, Matplotlib, Seaborn

## 🤖 Model Performance
* **Algorithm Used:** Random Forest Classifier
* **Accuracy Achieved:** 99.32%
